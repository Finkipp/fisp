Руководство пользователя программы управления техникой
Введение:
Добро пожаловать в программу управления техникой uchetoteh! Это приложение предназначено для учета и контроля состояния оборудования. С помощью данной программы вы сможете добавлять новые записи о технике, обновлять их статус, просматривать список всех единиц техники и создавать отчеты.

Начало работы:
Перед началом использования программы необходимо создать базу данных. Для этого выполните следующие шаги:
1.	Запустите программу.
2.	Выберите пункт меню Файл > Создать базу данных.
3.	После создания базы данных вы можете приступить к работе с программой.

Основные функции:
Добавление новой единицы техники
Для добавления новой записи о единице техники выполните следующее:
1.	Выберите пункт меню Техника > Добавить технику.
2.	В появившемся окне введите наименование техники.
3.	В следующем окне выберите тип техники.
4.	Затем укажите производителя техники.
5.	Далее введите дату покупки техники.
6.	Укажите срок окончания гарантии.
7.	Наконец, выберите текущий статус техники.
8.	Нажмите кнопку OK, чтобы сохранить запись.
Обновление статуса техники
Чтобы изменить статус существующей единицы техники, сделайте следующее:
1.	Найдите нужную строку в таблице с техникой.
2.	Щелкните правой кнопкой мыши на строке и выберите пункт Обновить статус.
3.	Во всплывающем окне введите новый статус техники и нажмите ОК.
Имеется также анологичный способ для старых версий программ, он заключается в закрытии окна программы и повторного её запуска. Если данная процедура не помогает, то рекомендуется перезагрузить устройство.
Просмотр списка техники
Список всей добавленной техники отображается в основной таблице приложения. Вы можете сортировать данные по различным столбцам, щелкая на заголовках столбцов.
Создание отчета
Для генерации отчета по всем имеющимся данным о технике выполните следующую последовательность действий:
1. Перейдите в меню Файл.
2. Выберите команду Печать (Создать отчет). Вы также можете нажать комбинацию Ctrl+P на клавиатуре, чтобы сразу открыть данное окно.
3. Программа автоматически создаст файл отчета и предложит вам открыть его.
